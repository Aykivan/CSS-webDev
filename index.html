    Псевдоклассы для стилизации ссылок, одно двоеточие:
  • hover - срабатывает при наведении
  • visited - работает только с сылками, после первого посещения
  • active - работвет при нажатии, можно применять к любым активным эл-там
  • focus - работает при выделении ссылки клавишей Tab
    Псевдоэлементы для стилизации фрагментов текста, два двоеточия:
  • first-letter - работа с первым эл-том

  • selection - работа с текстом при его выделении, не влияет на размер

  • after - позволяет работать с областью после обьектом

  • li:first-child - первый элемент

    • even - четные эл-ты
    • odd - не четные

    • №Эл.n - каждый энный эл-т (ее номер + n)
      • Так же можно ввести в скобках + n, число от которого начнется отсчет.

2. Каскадность и наследование

Существуют ряд правил для применения стилей.

  • Если эл-ты имеют одинаковый вес, то преобладать будет тот, что указан ниже в списке каскадных таблиц - CSS.
  • К эл-ту будут применены те стили, специфичность которых больше
  • !important - позволяет гарантировать применение какого-либо стиля с данной пометкой. Однако это считается плохой практикой, которую лучше не применять.

Наследованность - это способность потомков, перенимать все примененные к их родителю стили. Не все стили являются наследуемыми.
Если же мы хотим присвоить значение стиля родителя к потомку, мы можем прописать тот показатель, который нам нужен и присвоить ему значение inherit. На практике же он применяется достаточно редко.

2.1 Браузерная верстка

У каждого браузера установлен ряд параметров, таких, как отступы, размер шрифта и т.д.
Для того, чтобы сделать ресурс кросплатформенным существует понятие - Браузерная верстка.

    Она включает в себя два основных подхода:
  • Normalize - все параметры, для всех браузеров задаются одинаковыми.
  • Reset - все установленные по умолчанию пар-ры сбрасываются.

Для того, чтобы использовать один из этих принципов, неоюходимо счать стили для браузеров и установить их отдельныи файлом, добавив ссылку на них перед ссылками на остальные стили в HTML.

    Свойства: position
  • static - значение по умолчанию
  • relative - относительное позиционирование, эл-т формирует новый поток и его можно двигать
  • absolute - основной поток теряет связь с элементом и его место заполняется
  • fixed - позиция всегда будет отсчитываться от края окна браузера
  • sticky - применяется очень редко так как не поддерживается всеми браузерами
    Размеры:
  • widtd - ширина
  • height - высота
    • При данных параметрах блок не меняет своих размеров и не подстраивается под контект. Однако если мы придадим данным значениям приставки:

  • max-height - задает максимальное значение на которое эл-т может растянутся

Существуют два вида шрифтов, с засечками(serif) и без них (sans-serif) при применении шрифта, всегда указывается один из вариантов написания, так как если на компьютере не будет указанного шрифта был использован любой доступный на ПК с засечками или без них.

Так же существует два способа задать шрифт, выше представле способ с использование ссылки на шрифт. Так же мы можем скачать шрифт на компьютер и использовать относительные ссылки. Ниже в CSS есть комментарий с описанием подключения установленных шрифтов, при его использовании необходимо указывать формат шрифта, разные браузеры используют разные форматы, а значит для кросс браузерной верстки нужно указать их все.

    Поговорим остилизации текста:
  • color - задает цвет текста
  • text-decoration - подчеркиваание
    • underline - под текстом
    • overline - над текстом
    • line-through - зачеркнутыйы текст
    • none - убирает все подчеркивание
  • letter-spacing - расстояние между символами
  • word-spacing - расстояние между словами
  • line-height - межстрочный интервал
  • text-transform:
    • uppercase - все буквы - большие
    • lovercase - все буквы маленькие
    • capitalize - каждая первая буква большая
    • none
  • text-align - выравнивание текста
    • center
    • left
    • right
    • justify - равномерно по центру
  • text-indent - задает красную строку
